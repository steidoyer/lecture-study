# 강의 예제 코드 기록 저장소

## 시작하기 전

- 예제 코드를 다시한번 스스로 작성해보면서 공부하는 저장소입니다.
- 학습 필요에 따라 코드를 강의에 나온 그대로 작성하지 않고 변형할 수 있습니다.
- 예제 코드의 업로드 가능여부 확인 후 올리지만 혹 업로드가 불가한 코드가 있다면 언제든 삭제 가능하니 말씀해주세요.

## 저장소 설정

### gitignore

- https://www.toptal.com/developers/gitignore 에서 `react`, `windows`, `visual studio code` 조건으로 생성
